Transformer Enhanced Language Translator

NLP | Python | PyTorch

Built a sophisticated Transformer-based model using PyTorch for English-to-Italian text translation. The project focused on optimizing dataset preprocessing, applying tokenization techniques, and creating input-output pairs for training. By implementing essential Transformer components such as multi-head attention and positional encoding, the model achieved a high level of context understanding, significantly improving the accuracy of translations.

Activity Diary

HTML5 | CSS | React.js | Node.js | MongoDB

Developed a full-stack web application for task management. The app enables users to add, mark complete, or cancel tasks via a seamless, user-friendly interface built with React.js. Tourism Management System

Java | JDBC | MySQL

Created a Java-based terminal application for managing tourism services. The system allows users to browse popular tourist destinations, search for available tours, and book tickets. Integrated with MySQL through JDBC, the application provides robust data management, storing user details and bookings. Features include generating and printing booking confirmations and receipts, enhancing the user experience.
